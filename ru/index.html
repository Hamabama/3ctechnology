Наша Компания путем долгих исследований создала технологию получения сухой смеси ячеистого бетона из которой производится неавтоклавный газофибробетон Composite Cellular Concrete (далее «3С»). Эта технология позволяет просто размешать сухую смесь в воде и получить ячеистый бетон гарантированного качества. Газофибробетон из сухой смеси «3С» можно приготовить на любом существующем оборудовании для получения сухих смесей, бетона, штукатурки и т.д., включая ручную дрель. Работа с сухой смесью «3С» - проста и понятна, не требует специального обучения.

Оборудование для Приготовления «3С»

Сухая смесь газофибробетона «3С» готовится практически на любом существующем оборудовании, предназначенном для производства строительных смесей, штукатурок, бетонов, наливных полов и т. д.

Для приготовления на объекте необходимым условием является наличие воды и электроэнергии.

При больших объемах сухая смесь доставляется на объекты в силосах и заливается штукатурной станцией, либо доставляется в бетоносмесителях, предназначенных для доставки тяжелого бетона.

При малых объемах сухая смесь размешивается в емкости простой дрелью! Технология «3С» позволяет использовать любое смесительное оборудование.
